#   Aplicativo de gestão de vinhos Vinheria Agnello Mobile

##  Informações do Projeto

*   Linguagem de programação Kotlin.
*   Biblioteca Room para persistência de dados.
*   Jetpack Compose para interface gráfica.
*   Lógica de negócio está acoplada diretamente às telas (funções Composable).
*   Utilização de State Hoisting.
*   Banco de dados SQLite
*   Navegação entre telas gerenciada pelo Jetpack Navigation for Compose.


##  Instruções para rodar o projeto

1.  Clone o repositório e abra no Android Studio.
2.  Espere o Gradle sincronizar.
3.  Inicie um emulador de celular Android.
4.  E rode a aplicação.

## Funcionalidades 

*   Adicionar Vinho
*   Listar Vinhos
*   Editar Vinho
*   Deletar Vinho

## Jornada do Usuário

*  Assim que o aplicativo abrir, clique no botão Adicionar vinho e insira os dados do vinho. Depois clique em cadastrar e o vinho será adicionado.
*  Logo após, clique no botão voltar, e depois no botão Lista de vinhos.
*  Será possível ver as informações do vinho cadastrado. 
*  Para editar, clique no ícone "lápis" e susbtitua novas informações do vinho, depois clique no botão Atualizar.
*  Para excluir o vinho, clique no ícone de lixeira.
